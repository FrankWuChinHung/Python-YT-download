# Python-YT-download
利用pytube，moviepy套件來下載 You Tube 影片
並且使用flask讓使用者介面與結果呈現在網頁上，最後使用AWS架設網際網路端，網址:http://52.194.220.146:5000/

以下是.py檔案與.html檔案的目錄位置
