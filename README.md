# Python-YT-download
利用pytube，moviepy套件來下載 You Tube 影片
並且使用flask讓使用者介面與結果呈現在網頁上

以下是.py檔案與.html檔案的目錄位置

C:\Users(使用者)\your computer name (例如Frank)\WEB_APP

  YT_download_app.py
  
  templates
  
  static
  
C:\Users\Frank\WEB_APP\templates

  index.html
  
  download.html

C:\Users\Frank\WEB_APP\static

  bootstrap.min.css
  
  bootstrap.min.js
